# Ping Pong Game
這是一個簡單的乒乓球遊戲

這個遊戲的主要目的是為了當作我們的自主學習研究內容

## 安裝

 1. 使用`npm install`安裝
 2. 執行`node main.js`以啟動伺服器端
 3. 在劉覽器端開啟`http://(Your Host):9487`以開啟遊戲戶用端
